## 👩‍💻 Dev

### 1. 📎 Clonar el repositorio:

```sh
git clone https://github.com/CMRicardo/restaurant
cd restaurant
```

### 2. ⬇️ Instalar las dependencias
```sh
npm install
```

### 3. ⬆️ Levantar el servidor de desarrollo
```sh
npm start
```

## 💡 Recomendaciones

- Angular Language Service
